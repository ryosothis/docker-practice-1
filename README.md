# Трекер привычек - Docker проект
## Выполнили: Опёнков Всеволод и Савельев Святослав

## Описание проекта
Простой HTTP API для отслеживания выполнения привычек. Позволяет создавать привычки, отмечать их выполнение по дням и получать текущую серию (streak) выполнения.

## Функциональность
- Создание новой привычки
- Отметка выполнения привычки за определенную дату
- Получение текущей серии выполнения (количество дней подряд)
- Просмотр всех созданных привычек

## Технологии
- Python 3.10
- Flask (веб-фреймворк)
- Docker & Docker Compose
- In-memory storage (хранение данных в списках)

## Запуск проекта

### Требования
- Docker Desktop установлен и запущен
- Порт 5000 не занят

### Инструкция по запуску
1. Клонируйте репозиторий
2. Перейдите в папку `Openkov_Savelyev`
3. Запустите контейнер docker compose up --build
4. API доступно по адресу: http://localhost:5000
